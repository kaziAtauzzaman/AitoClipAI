# AitoClipAI

AitoClipAI is a new Python project. This repository currently contains only the
production-ready project structure and supporting configuration files.

No application logic has been implemented yet.

## Project Structure

- `src/` - Python source package code will live here.
- `tests/` - Automated tests will live here.
- `docs/` - Project documentation will live here.
- `config/` - Configuration files and examples will live here.
- `data/` - Local runtime data, generated assets, and logs.
- `models/` - Model files, checkpoints, and metadata.
- `scripts/` - Developer and operational helper scripts.

## Development

Install dependencies once they are added:

```bash
pip install -r requirements.txt
```

Run tests once test suites are added:

```bash
pytest
```
