# scripts

Developer, maintenance, and operational helper scripts will live in this
directory.

Scripts should be documented and kept small enough to audit easily.

`validate_prerecorded_pipeline.py` is a narrow developer harness for Pipeline
Validation 0.1. It processes one prerecorded URL or local file into run-scoped
analysis, rendering, logs, and a deterministic validation report. It is not the
general AitoClipAI application CLI.
